Class AuthStateService

AuthStateService is a service responsible for managing the authentication state of the user in the application. It maintains the current user and the associated authentication token. User and token data are stored in local storage to persist state across browser sessions.

The service provides methods for setting, getting, and resetting the user and token. It also helps to get authorization headers for HTTP requests, and get the favourite movie IDs of the current user.
